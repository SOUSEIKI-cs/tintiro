# TINTIRORIN
これはチンチロリンをプレイできるWEBサイトです。<br/>このゲームではチンチロリンと闇のゲームの人権ポイント式HPシステムを融合させたローカルルールを採用しています。

## 実行方法
このリポジトリをクローンした後に、`npm run dev`を実行して下さい。

## ゲームプレイ
1.  初期画面でプレイする人数を選択します。上限は10名です。(一緒ににダイスを振る順番を決めておいてください。)
2.  "GO"ボタンを押し、ゲームを始めます。
3.  最初にダイスロールフェーズが始まります。このフェーズでは先ほど決めた順番でサイコロを振って役を作ります。プレイヤーは全員3回づつ振ることができます。
4.  プレイヤーは"ドロー"ボタンを押してサイコロを振ります。保存する出目が決まったもしくは3回振ったら、"ターンエンド"ボタンをおしてターンエンドします。同じように手順3,4を全員分行います。
5.  全員振り終わったら、ダイレクトアタックフェーズに入ります。このフェーズでは出目のレベルが最も高いものが勝者となり、敗者にダメージを与えます。勝者は"執行"ボタンをおして敗者にダイレクトアタックをしましょう。
6.  ダイレクトアタックが終了すると、またダイスロールフェーズに戻り、サイコロを振ります。このあとは手順3~5を生きている人が一人になるまで繰り返します。
7.  生き残った一人がこのゲームを制した魔王です。

### ダメージについて
ダイスの出目にはレベルが存在し、レベルが高いほどダイレクトアタックフェーズで強くなります。あなたが敗者となったとき、勝者とのレベルの差*500のダメージを受けます(初期HPは6000です)。
<br/>出目は順不同です。勝者とレベルが同じ場合はダメージを受けません。
|出目|名前|レベル|
|--|--|--|
|1,1,1|ピンゾロ|6|
|2,2,2~6,6,6|ゾロメ|5|
|4,5,6|シゴロ|4|
|1,2,1のように、<br>2つの数字が同じのとき|役あり|3|
|2,5,6のようにすべて違う出目<br>(シゴロとヒフミを除く)|役なし|2|
|1,2,3|ヒフミ|1|
